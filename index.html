Sistema de Gestão de Treinamentos

Automatize e otimize a gestão de treinamentos na sua empresa.

Sobre o Projeto

Este projeto visa desenvolver um sistema que automatize e otimize o controle e a administração dos treinamentos de funcionários. O sistema permitirá aos gestores cadastrar funcionários, identificar automaticamente os treinamentos necessários para cada funcionário com base no setor, acompanhar o progresso dos treinamentos e manter um registro completo das atividades de treinamento.

Com a implementação deste sistema, espera-se que os gestores possam gerenciar os treinamentos de maneira mais eficaz, garantindo que todos os funcionários recebam a qualificação necessária de forma oportuna.

Funcionalidades

Cadastro de Funcionários

Registre funcionários e seus setores facilmente.

Identificação de Treinamentos

Determine automaticamente os treinamentos necessários.

Acompanhamento

Monitore o progresso dos treinamentos em tempo real.

Notificações

Receba alertas sobre a conclusão de cursos.

Dashboard

Visualize os treinamentos concluídos por setor.

Proposta

Objetivo

Desenvolver um software que permita aos gestores de treinamento cadastrar funcionários, identificar automaticamente os treinamentos necessários, acompanhar o progresso e manter um registro completo das atividades de treinamento.

Escopo

  • Cadastrar funcionários e setores
  • Identificar treinamentos necessários automaticamente
  • Monitorar o progresso dos treinamentos
  • Manter registros completos das atividades
  • Notificar responsáveis pela conclusão de cursos
  • Fornecer um dashboard de visualização

Soluções Propostas

  • Banco de dados relacional MySQL
  • Interface web responsiva com HTML, CSS e JavaScript
  • Lógica de negócios em Python para automação
